# Admin Review Prompts

## Canon Review Prompt

Review this Canon Change Request.

Check:

1. Is the source evidence clear?
2. Was the decision actually made?
3. Who had authority?
4. Does this conflict with existing canon?
5. Does it need circle/steward review?
6. Should it be approved, rejected, clarified, or escalated?

Return:

- Recommended status
- Reason
- Suggested revised language
- Implementation destination
- Any warnings

## Memory Review Prompt

Review this Memory Candidate.

Check:

1. Is it useful?
2. Is it sourced?
3. Is it neutral?
4. Does it avoid blame?
5. Does it belong in memory instead of canon?
6. Is it sensitive?

Return:

- Recommended status
- Suggested rewrite if needed
- Destination
- Risk if approved
- Risk if ignored

## Decision Review Prompt

Review this Decision Candidate.

Check:

1. Was this a real decision?
2. Is it only a suggestion?
3. Who made it?
4. Does that person/circle have authority?
5. Does it require canon update?
6. Are implementation tasks needed?

Return:

- Confirmed / Rejected / Needs Clarification / Escalated
- Reason
- Suggested next action
