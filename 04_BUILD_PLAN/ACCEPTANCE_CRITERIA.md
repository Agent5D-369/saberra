# Acceptance Criteria

## MVP is complete when:

1. Roots inbox can be polled.
2. New unprocessed emails are found.
3. Emails are classified.
4. Source Email records are created in Notion.
5. Google Meet asset links are parsed.
6. Meeting records are created/updated.
7. Meeting Asset records are created/updated.
8. Drive/Docs access is checked.
9. Access failures create Needs Access status.
10. Access request email sends.
11. Accessible Google Docs export to text.
12. Claude API extracts structured JSON.
13. JSON is validated.
14. Invalid JSON is retried or routed to manual review.
15. Tasks are created from clear action items.
16. Decision Candidates are created.
17. Risks are created.
18. Memory Candidates are created.
19. Canon Change Requests are created for canon-impacting content.
20. Sensitive flags route to review.
21. Processing Events log all runs.
22. Duplicate emails/assets do not duplicate records.
23. Staff workflow does not require logging into Roots.
24. Automation does not publish canon.
25. Deployment runs on Railway.
26. Rick has step-by-step setup instructions for Claude Team.
