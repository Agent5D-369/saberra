# User-Facing Manual Summary

## Everyday Name

Amora Living Memory Hub

Short name:

Living Memory

## Simple Rule

Invite Roots to memory-worthy meetings. Copy or forward Roots on memory-worthy emails.

## Meetings

1. Schedule the meeting normally.
2. Add `roots@amora.cr` as a guest.
3. Turn on recording/transcript/notes when appropriate.
4. Name decisions, owners, and next steps clearly.
5. Let the system process outputs after the meeting.
6. If Roots was not invited, forward asset emails and share files with Roots.

## Emails

1. CC Roots when the email should visibly enter project memory.
2. BCC Roots when capture is appropriate but should not add another visible recipient.
3. Forward important existing threads to Roots.
4. Use `[AMORA CAPTURE]` if helpful.
5. Do not copy Roots on everything.

## What Gets Captured

- Governance
- Roles
- Circle structure
- Tasks
- Decisions
- Land/legal/finance conversations
- Project planning
- Important stakeholder commitments
- Policies
- Risks
- Tensions

## What Does Not Need Capture

- Casual check-ins
- Private emotional processing
- Early brainstorming with no action
- Personal conversations
- Sensitive sharing without consent

## Team Promise

Living Memory is not surveillance. It is shared remembrance and coordination.
